首先创建需要上传的文件夹并切换到该文件夹下：
mkdir filename
cd filename
然后初始化库：
git init
添加远程库：
git remote add origin https://github.com/me/repo.git  //这里的me指的是github的账户名，repo指的是github上的仓库名

以后上传时按照以下步骤进行：
1. git add . // add all documents to暂存库
2. git commit -m "first commit"
3. git push origin master, push之前先pull一下



有一次出现  git everything up-to-date
solution:
1. git remote rm origin
2. git gc
然后重新重复上述操作步骤


有一次pull时出现 fatal: refusing to merge unrelated histories 
solution：
git pull origin master --allow-unrelated-histories
