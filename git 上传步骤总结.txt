首先创建需要上传的文件夹并切换到该文件夹下：
mkdir filename
cd filename
然后初始化库：
git init
然后开始上传：
1. git add . // add all documents to暂存库
2. git remote add origin https://github.com/me/repo.git
这里的me指的是github的账户名，repo指的是github上的仓库名
3. git commit -m "first commit"
4. git push origin master, push之前先pull一下



有一次出现  git everything up-to-date
solution:
1. git remote rm origin
2. git gc
然后重新重复上述操作步骤
